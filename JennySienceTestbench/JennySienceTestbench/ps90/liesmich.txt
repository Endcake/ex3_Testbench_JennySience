Um die Funktionen in Ihrer Anwendung zu benutzen, beachten Sie folgendes:
1. Die Datei "ps90func.cs" in das Projekt einbinden.
2. Die Bibliothek "ps90.dll" in das Projekt als Datei einbinden.
Bitte beachten, dass sie eine C-Bibliothek, und keine C#-Bibliothek(Assembly) ist.
3. Projekt-Eigenschaften ändern (Reiter "Erstellen(Kompilieren)", Option "Unsicheren Code zulassen" aktivieren).
4. Namespace "ps90func" einbinden ("using ps90func;").

Um die Demoanwendung (64-Bit) nutzen zu können, sind die folgenden Dateien erforderlich:
- Visual C++-Laufzeitdateien
Diese Dateien können als erforderliche Komponenten während der Installation auf einem Computer ausgefuehrt werden, auf dem Visual C++ 2010 Express Edition (oder hoeher) nicht installiert ist.
vcredist_x64.exe - Redistributionspaket fuer Visual C++ (64-Bit)
